# 🎮 GameCore | Game Project Manager

A local project manager built for indie developers to keep ideas, assets, tasks, notes, and progress in one intuitive workspace.

🚧 **This project is currently in active development.**

---

## ✨ Features (Planned / In Progress)

* 🧠 Manage game ideas and concepts
* 📁 Organize assets (images, audio, scripts, etc.)
* ✅ Track tasks and development progress
* 📝 Take structured notes per project
* 📊 Visualize project progression
* 💾 Fully local storage (no cloud required)

---

## 🧱 Tech Stack

*  **Core Language:** Rust (backend) • TypeScript (frontend)
*  **UI Layer:** React 19 + Tauri WebView2
*  **Data Storage:** Local filesystem (JSON -> serde_json / SQLite planned)
*  **Build System:** Vite (frontend) • Cargo (Rust backend)
*  **Architecture:** Desktop app (Tauri 2), offline-first, modular

---

## 🚀 Project Status

This application is still under development.

Current focus:

* Core project structure
* Local data management system
* Basic UI prototype
* Task & project models

---

## 📂 Goal

The goal of this tool is to provide indie developers with a **lightweight, fast, and offline-first workspace** to manage all aspects of game development in one place.
